NOTE: The RPGMaker web player has known issues in Firefox. I have attempted to manually patch around some of the loading issues, but sometimes the title screen still loads without a background image; it should still be possible to start the game without issues in this case. If you still have difficulty starting the game, try holding shift while you refresh the page (this clears the cache before refreshing, which allows the player to load more reliably). Other browsers do not appear to have these issues, so if you are having too much difficulty, you may have an easier time playing the game in another browser.
